# 📙 Module C — Power BI Executive Dashboard

## 📌 Objective
To design an executive-level dashboard that consolidates financial performance, segment analysis, and fraud insights into a single, decision-ready interface.

---

## 📊 Dashboard Overview

This dashboard presents a comprehensive view of:

- Revenue growth and forecasts
- Segment-level performance
- Profitability trends (EBITDA & margins)
- Transaction behavior and fraud detection gaps

---

## 🧠 Approach

### 1. Data Integration
- Combined outputs from:
  - Financial modeling (Module A)
  - SQL transaction analysis (Module B)
- Cleaned and transformed data using Power Query

### 2. Data Modeling
- Established relationships between financial and transaction datasets
- Created calculated measures using DAX:
  - Revenue growth %
  - EBITDA margins
  - Forecast scenarios

### 3. Dashboard Design
- Built KPI cards for high-level metrics
- Designed trend visualizations for performance tracking
- Structured layout for executive readability

---

## 📈 Key Features

### 🔹 Financial Performance
- Total revenue and YoY growth
- Revenue breakdown by segment
- Forecast scenarios (Base Case vs Bear Case)

### 🔹 Profitability Analysis
- EBITDA by segment
- Margin recovery trends
- Identification of most profitable business unit

### 🔹 Fraud & Transaction Insights
- Transaction volume by type
- WoW Volume Growth %
- Busiest Transaction Hours
- Fraud occurrence vs system detection
- Identification of major detection gaps

---

## 🔍 Key Insights

- Revenue grew significantly, driven primarily by Deliveries
- Mobility remains the most profitable segment
- Financial Services is improving but still in a growth phase
- ⚠ Fraud detection system flagged only a small fraction of actual fraud cases, indicating a critical operational risk
