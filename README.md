# Decoding Grab: A Southeast Asian Fintech Deep Dive 
A three-module deep-dive into Grab Holdings (Southeast Asia's super-app) modelling its segment revenue structure, analyzing transaction risk and behavior via SQL on a synthetic payments dataset, and synthesizing insights into an executive Power BI dashboard.

## 📈 Key Insights
- Revenue grew 20.5% YoY driven by Deliveries
- Mobility remains the most profitable segment
- Financial Services is high-growth but still unprofitable
- Fraud detection system flagged only 0.2% of actual fraud cases
